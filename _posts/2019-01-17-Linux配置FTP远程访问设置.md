## 检测是否安装vsftpd

```
vsftpd -version
```

## 安装vsftpd

```
sudo apt-get install vsftpd
vsftpd -version
```

## 新建ftp共享文件夹

```
mkdir /home/ftp
```

## 创建ftp用户名密码

```
sudo useradd -d /home/ftp -s /bin/bash myftp
passwd myftp
```

## 配置文件/etc/vsftpd.conf

vi vsftpd.conf可以对访问权限等进行设置。

## 启动服务

```
service vsftpd start
```

## 测试ftp

利用ifconfig -a命令查看linux的ip地址，然后可以在局域网下，通过浏览器输入ftp://ip地址进行访问。
